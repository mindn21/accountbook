# 국환♥혜지 가계부 📱💰

위플 가계부 스타일의 2인용 가계부 웹앱입니다.

## ✨ 주요 기능

### 💸 입출금 관리
- 사용자별 (국환/혜지) 수입/지출 기록
- 카테고리별 분류 (식비, 카페, 교통, 쇼핑 등)
- 결제수단 선택 (현금, 체크카드, 신용카드 등)
- 자산 선택 (입출금 통장, 자동차 통장, 생활비 통장)
- 금액 빠른 입력 버튼 (+10만, +1만, +1천, +1백)

### 🏦 자산 관리
- 여러 통장/자산 관리
- 자산 간 이동 기능
- 자산별 실시간 잔액 표시

### 📊 거래 로그 & 필터링
- 모든 거래 내역 시간순 정렬
- 유형별 필터 (전체/수입/지출/자산이동)
- 날짜별 그룹화 표시

### 📈 통계
- 사용자별 지출 비교 (국환 vs 혜지)
- 카테고리별 지출 분석
- 결제수단별 지출 분석 (사용자별 세부 금액 표시)

### ⚙️ 설정
- 결제수단 추가/수정/삭제
- 자산 추가/수정/삭제
- 카테고리 추가/수정/삭제
- 즐겨찾기 설정 (자주 쓰는 항목만 표시)
- 표시 순서 변경 (↑↓ 버튼)

## 🚀 사용 방법

### 1. 로컬에서 테스트
1. `index.html` 파일을 브라우저로 엽니다
2. 자동으로 로컬 저장소 모드로 실행됩니다
3. 데이터는 브라우저의 localStorage에 저장됩니다

### 2. 아이폰에서 앱처럼 사용
1. Safari에서 `index.html` 열기
2. 하단 공유 버튼 탭
3. "홈 화면에 추가" 선택
4. 이제 앱처럼 사용 가능!

### 3. Supabase 연동 (선택사항)
두 사람이 데이터를 공유하려면 Supabase를 연동하세요.

#### Supabase 설정
1. [supabase.com](https://supabase.com)에서 무료 계정 생성
2. 새 프로젝트 생성 (서울 리전 권장)
3. SQL Editor 열기
4. `supabase-setup.sql` 파일의 내용을 복사해서 실행
5. Settings > API 메뉴로 이동
6. `Project URL`과 `anon public` key 복사

#### 앱에서 연결
1. 설정 탭으로 이동
2. Supabase URL과 Key 입력
3. "연결하기" 버튼 클릭
4. 연결 상태가 "Supabase 연결됨"으로 변경되면 성공!

이제 두 폰에서 같은 계정으로 연결하면 데이터가 실시간 동기화됩니다.
